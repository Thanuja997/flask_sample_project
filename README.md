## Installation


Create a New Environment.

```bash
python3 -m venv env
```


Install dependencies.

```bash
pip install -r requirements.txt
```


## Usage

Start the server.

```bash
python3 -m flask run
```
Ctrl+click on the link that appears in terminal. ( Normally the link will be [http://127.0.0.1:5000/](http://127.0.0.1:5000/)  )
